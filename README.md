# ChinesePoetrySpider
中文古诗词爬虫，简体全唐诗数据，用于学习研究，不进行任何商业用途，如有侵权请告知删除！

### 全唐诗

1. 字段

* id ： 序号
* pid ：唯一标识符md5（作者+诗词名+卷号+序号）
* title：诗词名
* author：作者
* pageId：卷号
* page：卷名
* pageNumber：卷下面的序号
* dynasty：朝代
* Lang：语言（简体/繁体）
* content：内容
* hour：时辰（早中晚）
* season：季节（春夏秋冬）
* weather：天气
* location：地点
* sentiment：情感
* type：类型（边塞诗，田园诗等）
* character：包含的人物（默认为空）
* fromBy：出处
* Url：链接
